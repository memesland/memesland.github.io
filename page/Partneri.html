Наши партнеры


TJournal — новое медиа

О сайте на нем самом сказано: «Мы пишем статьи и программируем роботов, которые анализируют социальные сети. Затем собираем самое важное на страницах проекта. Наши читатели получают информацию быстрее и точнее остальных.»


GitHub - хостинг сайтов и IT проектов

Крупнейший веб-сервис для хостинга IT-проектов и их совместной разработки. Сервис абсолютно бесплатен для проектов с открытым исходным кодом и предоставляет им все возможности, а для частных проектов предлагаются различные платные тарифные планы.


ВКонтакте - социальная сеть

Это социальная сеть для быстрой и удобной коммуникации между людьми по всему миру. Задача ВКонтакте — в каждый отдельно взятый момент оставаться наиболее современным, быстрым и эстетичным способом общения в сети.


Telegram - мессенджер

Telegram — бесплатный кроссплатформенный мессенджер для смартфонов и других устройств, позволяющий обмениваться текстовыми сообщениями и медиафайлами различных форматов.


Font Awesome - иконочный шрифт

Вам не придется тратить время на файлы на вашем сервере. Вы получите все 675 иконки плюс наборы стилей - все оптимизировано для быстрой загрузки.
